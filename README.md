# 2D Adventure

**Game Description**  
2D Adventure is a top-down action RPG written in Java. The game features dynamic combat elements, map exploration, and character progression.

---

## 🌟 Features
Currently implemented:
- **NPCs and main character**
- **Explorable map**
- **Collision detection**
- **Sound effects**

### ✅ Planned Features:
1. **Dialogue window with NPCs**
2. **Events**

4. **Main screen**
5. **Monsters and bosses**

7. **Cutscenes**
